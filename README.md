# QA_Robot

#### 介绍
本项目是大三上自然语言处理课程项目，基于seq2seq模型，实现简易的对话机器人效果。

#### 安装教程

```
git clone https://gitee.com/fish-in-jiangan-river/qa_-robot.git
```


#### 使用说明

```
//训练（电脑配置不高时，建议在云GPU进行训练）
python main.py -m train



//闲聊
python main.py
```
